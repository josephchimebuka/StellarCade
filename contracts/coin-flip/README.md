# Coin Flip Contract

## Overview

The Coin Flip contract implements a simple, fast-paced betting game.

## Gameplay

1. Choose Heads (0) or Tails (1).
2. Place your bet.
3. The contract generates a provably fair result.
4. If you win, you receive ~2x your bet (minus fee).

## Integration

- **Prize Pool**: Uses the Prize Pool for asset escrow and payouts.
- **RNG**: Uses the Random Generator for fair outcomes.
